# **App Name**: QbLog

## Core Features:

- Authenticated Workspace: Secure sign-in via GitHub and Google to provide a personalized experience and persistent data storage.
- Omni-Editor Snippet System: Create and manage code snippets with the Monaco Editor, featuring multi-language syntax highlighting and direct copy-to-clipboard actions.
- Structured Bug Tracker: Log problem/solution pairs for recurring bugs, allowing developers to trace technical debt and successful resolutions over time.
- Markdown Knowledge Engine: Advanced note-taking system supporting full Markdown formatting, code blocks, and folder-based hierarchical organization.
- AI Debugging & Refactoring Tool: An AI assistant tool that uses reasoning to analyze your snippets and debug logs to suggest optimizations or architectural fixes.
- Raycast-Style Spotlight Search: A rapid, keyboard-accessible command palette to instantly locate notes, snippets, or resources via fuzzy search logic.
- Scalable Mongo Storage: A high-performance document-based architecture leveraging MongoDB to ensure all technical documentation is persisted and queryable.

## Style Guidelines:

- Primary color: High-contrast Pure White (#FFFFFF) on a dark scheme. Background: Deep Onyx (#09090B), heavily desaturated and modern. Accent: Muted Zinc Gray (#71717A) for subtle depth and differentiation.
- Headline font: 'Poppins' for its geometric, high-end feel. Body font: 'Inter' for its technical clarity and legibility. Note: currently only Google Fonts are supported.
- Ultra-minimalist outline icons from the Lucide library, featuring consistent 1.5px stroke weight for a refined, surgical look.
- A rigid, clean-room dashboard featuring a high-precision sidebar and card-based modular layout with generous whitespace.
- Micro-interactions using Framer Motion with stiff spring curves for a snappy, high-performance interface feel during search and filtering.
